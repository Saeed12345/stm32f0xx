Lesson 1: Blink LED using Bare Metal Programming

In this lesson, we will learn how to blink an LED connected to pin PA5 on the STM32F0xx microcontroller using bare metal programming. We will include the header #include "stm32f0xx.h" in our code.

Watch the tutorial video:


// Include the STM32F0xx header
#include "stm32f0xx.h"

void delay(int a) {
    volatile int i, j;
    for (i = 0; i < a; i++) {
        j++;
    }
}

int main(void) {
    // Enable clock for GPIOA
    RCC->AHBENR |= RCC_AHBENR_GPIOAEN;

    // Set PA5 as output
    GPIOA->MODER |= GPIO_MODER_MODER5_0;

    while (1) {
        // Toggle PA5
        GPIOA->ODR ^= GPIO_ODR_5;

        // Delay
        delay(100000);
    }

    // We should never reach here
    return 0;
}
